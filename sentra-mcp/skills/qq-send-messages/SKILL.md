# QQ 消息发送（群聊/私聊）

## 适用场景
- 需要向 QQ 群或某个 QQ 用户发送消息。

## 工作流
1. 确认目标
   - 群聊：必须拿到 `group_id`
   - 私聊：必须拿到 `user_id`
2. 组织要发送的消息
   - `content` 应该是“最终要发送出去的文本”，保持简短、直接。
   - 不要为了显得真实而编造“我点进去看了某网站/某页面”。如需引用查询结果，只引用工具返回的要点。
3. 调用发送工具
   - 群：`local__send_group_message`
   - 私聊：`local__send_private_message`

## 参数/约束
- `group_id` / `user_id` 必须来自上下文或工具结果，不要使用占位符。
- `content` 是消息文本，不是计划，不需要把思考过程写进去。

## 常见错误与补救
- 目标 ID 不确定：先通过群/联系人相关工具获取，再发送。
- 内容过长或过度角色扮演：压缩为 1-6 行，保留关键信息。
